How NFC Tracking Works

Each NFC card has a unique URL programmed into it:

tapeo.co/go?d=DRIVER_ID&r=ROUTE&b=BUSINESS_ID

d = Driver ID (who has the card)

r = Route (airport, city_center, hotel_zone, etc.)

b = Business ID (if card is assigned to a specific business)

When a tourist taps → the server logs the tap with all metadata → redirects to the business website or the discover page.

To program a card: Use any NFC writing app (NFC Tools, TagWriter) and write the URL above with the right driver/route/business IDs.
